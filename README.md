<h2>Description</h2>

 - CalcVision is an application containing a 3D and 2D graphing calculator, allowing users to input any mathematical relation containing the variables x, y, and z, with the output displayed to the user via Unity’s particle system
 - The program contains pannable/zoomable 3D and 2D views that accept user input and display graphs using user chosen gradients to show the depth of each particle to help users visualize more complicated shapes
 - All assets were created in the Unity Game Engine, with scripting done via C# (algebraic parsing done via mXparser’s C# library)
